Instructions d'installation :
	- copier l'archive com.thoughtworks.xstream.jar dans le répertoire /usr/share/java/ si vous êtes sous Ubuntu
	- importer l'archive MiniProjet-Marongiu-Karassev.jar dans un projet eclipse
	- si vous êtes sous un autre système d'exploitation qu'Ubuntu, placer l'archive com.thoughtworks.xstream.jar dans le répertoire de votre choix et l'ajouter au buildpath de votre projet en prenant soin de supprimer celui spécifiant /usr/share/java/
	- exécuter la classe Main.java du package par défaut pour afficher la démonstration
